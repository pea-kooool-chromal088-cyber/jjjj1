# Руководство для администратора

## Как стать администратором

Чтобы получить права администратора для бота регистрации на мероприятия, необходимо:

1. Узнать свой ID в Telegram:
   - Отправьте любое сообщение боту
   - В логах консоли, где запущен бот, вы увидите запись с вашим ID:
     ```
     INFO:root:Received message from user: {'id': 123456789, 'is_bot': False, ...}
     ```
   - ID пользователя - это число после `'id':`

2. Добавить свой ID в файл `.env`:
   ```
   ADMIN_IDS=123456789
   ```
   
   Если администраторов несколько, перечислите ID через запятую:
   ```
   ADMIN_IDS=123456789,987654321,111222333
   ```

3. Перезапустить бота, чтобы изменения вступили в силу.

## Доступные команды администратора

После настройки ID и перезапуска бота, администратор может использовать следующие команды:

### /admin
Открывает админ-панель с возможностью быстрого доступа ко всем администраторским функциям.

### /stats
Показывает статистику регистрации:
- Общее количество зарегистрированных пользователей
- Количество созданных мероприятий
- Общее количество регистраций на мероприятия

### /new_event
Команда для создания нового мероприятия (функция в разработке).

### /events_list
Просмотр списка мероприятий (функция в разработке).

## Возможности администратора

Администратор имеет возможность:
- Просматривать статистику регистрации пользователей
- Создавать новые мероприятия (в будущих версиях)
- Просматривать список зарегистрированных пользователей (в будущих версиях)
- Управлять мероприятиями (в будущих версиях)

## Важные замечания

- ID администратора должен быть указан в точности как в Telegram
- После изменения .env файла обязательно перезапустите бота
- Не рекомендуется делиться своими правами администратора с другими лицами
